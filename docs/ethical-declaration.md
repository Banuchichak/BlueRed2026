# Etik və Hüquqi Bəyannamə

- Tətbiq yalnız **müdafiə (defensiv)** məqsədlərə xidmət edir — istifadəçiləri vishing
  hücumlarından qorumaq üçündür, heç bir hücum funksiyası ehtiva etmir.
- Etibarlı qurum bazasına yalnız təsdiqlənmə prosesindən keçmiş şirkət/qurumlar əlavə olunur.
- Səsli mesajlar və zəng tarixçəsi yalnız istifadəçinin razılığı ilə saxlanılır.
- Tokenlər vaxtaşırı avtomatik yenilənir ki, sızma halında köhnə token istifadə oluna bilməsin.
- MVP simulyasiyasında real telefon şəbəkəsi inteqrasiyası yoxdur — bütün zənglər
  brauzer daxilində simulyasiya olunur, real istifadəçi məlumatı toplanmır.
